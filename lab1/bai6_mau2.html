Two Column - Left Column Static

Ver 2.3

The left column is absolutely positioned and is a fixed width. It won't move.

The content column, which you are reading from, is flexible. It will automatically change shape, and the width can be controlled by setting appropriate margins or an actual width can be specified if required.

This template has been tested in:

  • Mozilla 1.5b & Firefox 0.8
  • Internet Explorer 6
  • Netscape 7.1
  • Opera 7.11
